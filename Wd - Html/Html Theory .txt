Q1 - Introduction to HTML and its structure
HyperText Markup Language,is the standard language used to structure and present content on the World Wide Web. 
It's the foundation of every webpage you see, providing the basic framework for how information is displayed.
Basic Structure:-
<!DOCTYPE html>
<html>
<head>
  <title>Html</title>
</head>
<body>
  <h1>Hello World</h1>
  <p>This is paragraph</p>
</body>
</html>  

Q2 - Explanation of key tags:
<a>: Anchor tag for hyperlinks.
<form>: Form tag for user input.
<table>: Table tag for data representation.
<img>: Image tag for embedding images.
<ul>, <ol>, and <li> : List tags
<p>: Paragraph tag.
<br>: Line break.
<label>: Label for form inputs.
